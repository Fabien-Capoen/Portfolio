Création d'un site internet pour l'établissement de formation

Contexte

Projet réalisé lors des séances d'atelier de professionnalisation.
Réalisé en 10 séances de 4 heures.

Notre responsable de formation nous à mendaté pour construire un site propre à létablissement.
le travail a été découpé en plusieurs taches réparties aux différents groupes

  • Chef de projet ( 1 personne )
  • Groupe de développement ( 3 personnes )
  • Groupe charte graphique ( 2 personnes )
  • Groupe création de contenu ( 4 personnes )

Pour ce projet, j'ai été le chef de projet.
J'ai été en charge de faire l'analyse des besoins, d'organiser les équipes, de commander les ressources nécessaire, et d'aider tous les groupes lorsqu'ils en avaient besoins.
Liste des ressources nécessaire :

  • Hébergement ( OVH )
  • Nom de domaine ( ufa47.info )
  • outil de développement ( Joomla )
  • Extensions Joomla ( "Template Creator CK" et "SP Page Builder"
